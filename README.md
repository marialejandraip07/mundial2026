# Copa Mundial FIFA 2026 ⚽🏆

Tracker interactivo para la Copa Mundial 2026 que se juega en EE.UU., Canadá y México.

## Características

- 📊 **Seguimiento de Grupos**: 12 grupos con 4 equipos cada uno
- 🏆 **Fase Eliminatoria**: 16avos hasta Final
- 💾 **Auto-guardado**: Tus datos se guardan automáticamente en localStorage
- 📱 **Responsive**: Diseño adaptado para móvil y desktop
- 🌈 **Colores por grupo**: Cada grupo tiene su propio color
- 🏳️ **Banderas**: Emojis de banderas para cada país

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm start
```

La app abrirá en [http://localhost:3000](http://localhost:3000).

## Deploy a GitHub Pages

```bash
npm run deploy
```

La app estará disponible en: https://marialejandraip07.github.io/mundial2026

## Build

```bash
npm run build
```

Crea la versión optimizada para producción en la carpeta `build/`.
